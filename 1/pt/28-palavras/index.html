Como Jogar

O Livro das 28 Palavras é uma aventura onde aprendes a ler e a escrever em 4 passos mágicos:

1. Descobrir: Raspa a imagem para veres a palavra nova.
2. Cortar: Bate palmas para dividir as sílabas e conhecer as famílias.
3. Construir: Junta os cartões corretos para formar novas palavras.
4. Escrever: Escreve tu mesmo a palavra secreta para ganhares!
